## 起因
豆瓣租房小组里面的信息比较杂乱：
1. 小组不能筛选， 导致多个城市里的租房信息混在一起
2. 一个帖子可以挂到不同小组，导致重复的帖子特别多  

这个爬虫主要解决以上两个痛点，并且得益于豆瓣的网站结构非常清晰，接口也简单。 感谢豆瓣

## 使用方法
1. 修改pipline 里的`InvalidGroupPipeline.pattern` 可以剔除城市关键字  
2. 修改`spider.QuotesSpider` 的place 为你想查找的小区等
3. 执行`python begin.py`
